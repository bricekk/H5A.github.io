Thanks for using h5assembler ♥

H5 assembler is an open source software under MIT license which allows html developers to no longer repeat their codes, it was designed to help html developers to easily integrate their html code into web frameworks such as laravel, vue js, django ,and others ... , easy to learn, and with its wide range of commands, it is the ideal tool for all developers who are embarking on large projects using only html and css. with H5assembler you will never need to repeat the same line of code several times
